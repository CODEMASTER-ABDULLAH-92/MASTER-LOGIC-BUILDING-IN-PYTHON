# 🧠 Logical Reasoning Master Logic in Python

Welcome to the **Logical Reasoning Master Logic** repository! This project is designed to help you build a strong foundation in **decision-making** using Python.  
We explore how to use **conditions**, **relational operators**, **logical operators**, and **nested if-else** statements to solve real-world logic problems step by step.

---

## 🎯 Objective
- Understand how to make decisions using conditions.
- Master the use of:
  - Relational Operators (`==`, `!=`, `>`, `<`, `>=`, `<=`)
  - Logical Operators (`and`, `or`, `not`)
  - Nested `if` and multiple conditions
- Build logical thinking through practical coding problems.

## 📝 Key Concepts Covered

### 🔹 Relational Operators
- `==` (equal to)
- `!=` (not equal to)
- `>` (greater than)
- `<` (less than)
- `>=` (greater than or equal to)
- `<=` (less than or equal to)

### 🔹 Logical Operators
- `and` - Both conditions must be True
- `or` - At least one condition must be True
- `not` - Reverses the boolean value

### 🔹 Conditional Statements
- `if` - Execute code if condition is True
- `elif` - Else if, checks another condition
- `else` - Execute code if all conditions are False

### 🔹 Nested Conditions
- `if` statements inside other `if` statements
- Used for checking multiple layers of conditions
